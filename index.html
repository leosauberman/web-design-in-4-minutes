Web Design em 4 minutos

por Jeremy Thomas

Traduzido por Leonardo Sauberman

Vamos dizer que você tem um produto, um portfolio ou simplesmente uma ideia que você quer compartilhar com todo mundo no seu próprio site. Antes de você publicar pra todo mundo na Internet, você quer que fique atrativo, profissional ou pelo menos decente.

Qual é a 1ª coisa que você precisa fazer?

Conteúdo

O propósito do design é incrementar a apresentação do conteúdo em que ele está sendo aplicado. Pode parecer óbvio, mas sendo o conteúdo o elemento primordial de um site, ele não ser pensado depois do design, mas sim antes.

Conteúdo escrito, como esse parágrafo que você está lendo agora, forma mais de 90% da Web. Estilizar esse conteúdo é um caminho longo a ser percorrido.

Vamos assumir que você já terminou o conteúdo que queria publicar e acabou de criar um arquivo style.css, qual é a 1ª regra que você pode escrever?

Centralizando

Linha longas podem ser difíceis de analisar e por isso difíceis de ler. Definir um limite de caractéres por linha melhora muito a legibilidade e apelo de um paredão de texto.

body {
    margin: 0 auto;
    max-width: 50em;
}

Depois de estilizar os blocos de texto, que tal estilizar o texto em si?

Fonte

A fonte padrão do navegador é "Times", o que não é nada apelativo (principalmente porque é a fonte "não-estilizada"). Mudar pra uma fonte sem serifa como "Helvetica" ou até "Arial" já melhora bastante o visual da sua página.

    body {
        font-family: "Helvetica", "Arial", sans-serif;
    }

Se você prefere uma fonte serifada, tenta a "Georgia".

Enquanto isso deixa o texto mais apelativo, vamos deixar mais legível também.

Espaçamento

Quando a página parece "quebrada" pro usuário, normalmente é um problema de espaçamento. Dar um espaço tanto ao redor quanto entre o conteúdo pode aumentar o apelo da sua página

    body {
        line-height: 1.5;
        padding: 4em 1em;
    }

    h2 {
        margin-top: 1em;
        padding-top: 1em;
    }

Agora que o layout já melhorou muito até agora, vamos aplicar umas mudanças sutis.

Cor & Contraste

Texto preto num fundo branco pode ser um pouco duro pros olhos. Optar por um tom mais fraco de preto pro texto faz a página ficar mais confortável de se ler.

    body {
        color: #555;
    }

E para manter um nível decente de contraste, vamos escolher um tom mais escuro pra palavras importantes 

    h1,
    h2,
    strong{
        color: #333;
    }

Quase toda a página já ficou visualmente melhor, mas alguns elementos (tipo os recortes de código) ainda parecem um pouco desconexos.

Equilíbrio

Só precisamos de alguns pequenos toques extras pra corrigir o equilíbrio da página:

    code,
    pre {
        background: #eee;
    }
    
    code {
        padding: 2px 4px;
        vertical-align: text-bottom;
    }
    
    pre {
        padding: 1em;
    }

Acho que a essa altura, você já queira que sua página se destaque e ganhe um pouco de identidade.

Cor Primária

A maioria das marcas tem uma cor primária que age como um toque visual. Em um site, esse toque pode ser usado pra enfatizar elementos interativos, como links.

    a {
        color: #e81c4f;
    }

Mas pra equilibrar, a gente vai precisar de algumas cores adicionais cores adicionais.

Cores secundárias

O tom secundário pode ser complementado com mais alguns tons sutis, a ser usado em bordas, plano de fundo e até em textos.

    body {
        color: #566b78;
    }

    code,
    pre {
        background: #f5f7f9;
        border-bottom: 1px solid #d8dee9;
        color: #a7adba;
    }

    pre {
        border-left: 2px solid #69c;
    }

Já que trocamos os tons, porque não trocar as formas...

Fonte Personalizada

Justamente pelo fato do texto ser o conteúdo principal de uma página web, usar uma fonte personalizada dá ainda mais uma identidade ao site.

Você pode usar sua própria fonte ou usar um serviço como Typekit, mas aqui vamos usar a "Roboto" do Google Fonts que é grátis.

    @import 'https://fonts.googleapis.com/css?family=Roboto:300,400,500';
    body {
        font-family: "Roboto", "Helvetica", "Arial", sans-serif;
    }

Depois de melhorar a identidade através do texto, que tal adicionar mil palavras...

Imagens e ícones podem ser usados como enfeites pra dar suporte ao seu conteúdo ou ser parte ativa da mensagem que você quer passar.

Vamos incrementar nosso header com uma imagem de fundo irada do Unsplash

    header {
        background-color: #263d36;
        background-image: url("header.jpg");
        background-position: center top;
        background-repeat: no-repeat;
        background-size: cover;
        line-height: 1.2;
        padding: 10vw 2em;
        text-align: center;
    }

Vamos adicionar, também, uma logo

    header img {
        display: inline-block;
        height: 120px;
        vertical-align: top;
        width: 120px;
    }

Vamos aproveitar essa oportunidade pra melhorar os estilos de texto.

    header h1 {
        color: white;
        font-size: 2.5em;
        font-weight: 300;
    }
    
    header a {
        border: 1px solid #e81c4f;
        border-radius: 290486px;
        color: white;
        font-size: 0.6em;
        letter-spacing: 0.2em;
        padding: 1em 2em;
        text-transform: uppercase;
        text-decoration: none;
        transition: none 200ms ease-out;
        transition-property: color, background;
    }
    
    header a:hover {
        background:  #e81c4f;
        color: white;
    }

E pronto!

A gente fez uma página decente em poucos minutos seguindo os princípios básicos do Web Design.
